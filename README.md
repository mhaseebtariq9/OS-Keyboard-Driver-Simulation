OS Keyboard Driver Simulation Project
   Project Description:
This project is a simulation of a keyboard driver in an Operating System.  
It demonstrates how keyboard inputs are detected, processed, and displayed in real time using a web-based interface.
   Features:
- Real-time keyboard input detection  
- Special key handling (Enter, Backspace, Shift, etc.)  
- Simple and interactive UI  
- OS-level behavior simulation using web technologies  
   Technologies Used:
- HTML  
- CSS  
- JavaScript
